# Alumni Portal Website

A modern alumni networking and engagement platform developed during my internship at **Cognexon Solutions**. This platform serves as an alternative digital community website for [IET Hub](https://iethub.org), enabling alumni and users to stay connected through blogs, news, and event management.

---

## 🚀 Live Demo

🌐 Website: (https://alumni2-phi.vercel.app/)


---

## 📖 Project Overview

The Alumni Portal is a full-stack web application designed to improve communication and interaction between alumni, students, and the organization. The platform allows users to:

- Read and publish blogs
- View latest news and announcements
- Explore upcoming events
- Register and attend events
- Stay connected with the alumni community

The project is deployed on Vercel and follows a modern MERN-based architecture with Next.js frontend support.

---

## ✨ Features

### 📰 Blogs Management
- Create and publish blogs
- Read blogs posted by alumni or admins
- Responsive blog pages with modern UI

### 📢 News & Announcements
- Post important updates and news
- Display latest announcements dynamically
- Easy content management

### 🎉 Events System
- Create and manage events
- Users can view upcoming events
- Event registration functionality
- Attendance tracking support

### 👤 User-Friendly Interface
- Clean and responsive design
- Mobile-friendly layout using Tailwind CSS
- Fast and optimized frontend with Next.js
